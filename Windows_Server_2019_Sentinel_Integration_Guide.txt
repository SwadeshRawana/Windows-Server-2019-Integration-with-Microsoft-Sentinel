## ðŸ“Š Part 4: Configure Log Collection in Sentinel

### âœ… Step 7: Create a Log Analytics Workspace (LAW)

2. Name: `Sentinel-LAW`
3. Select Resource Group and Region
4. Click **Review + Create**

### âœ… Step 8: Enable Microsoft Sentinel
1. Go to **Microsoft Sentinel > + Add**
2. Select your **LAW**
3. Click **Add**

### âœ… Step 9: Connect Data Source
1. In Sentinel, go to **Data connectors**
2. Search: `Windows Security Events via AMA`
3. Click **Open connector page**
4. Select your onboarded server
5. Choose logs:
   - Security
   - System
   - Application
6. Click **Apply changes**
